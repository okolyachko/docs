Get started
===========


.. raw:: html
  
  <p style="font-size: 18px">Want to create a Generative AI assistant? Follow the steps below to get started with the Alan AI Platform. </p>
   
1. Create an Alan AI Studio account
-----------------------------------

`Sign up for Alan AI Studio <https://studio.alan.app/register>`__, Alan AI's web-based IDE, to create a project for the AI assistant, test the dialog and customize your assistant.

.. raw:: html

   <div class="video-container" style="border-radius: 12px">

   </div>
   
|
   
2. Write a dialog script
------------------------

In Alan Studio, write a dialog script. Dialog scripts describe the structure of the dialog with the user, or the expected conversation scenario.

Get started swiftly:

- Go to :doc:`Dialog script concepts <server-api/script-concepts>` to learn what Alan AI's functionality you can use to create a dialog script.
- Check `Example scripts <https://github.com/alan-ai/voice-assistant-scripts>`__ on `Alan AI's GitHub <https://github.com/alan-ai>`__.

.. image:: _static/assets/img/about/get-started-scripts.png 

|
 
3. Integrate the AI assistant with your app
-------------------------------------------

Alan AI offers SDKs for different platforms, frameworks and languages to embed an AI assistant to your app. Select your platform to learn about :doc:`integration details <client-api/integration>`. Upon integration, you'll see a small draggable button on top of the app's UI.

4. Analyze and iterate
----------------------
	  
Alan AI provides you with advanced conversational analytics and success metrics through its :doc:`Logs <usage/guides/logs>` and :doc:`Analytics View<usage/guides/analytics-view>`. Use these tools to:

- Gain insight into the users' experience
- Analyze their behavior and flows
- Identify unhandled requests 
- Optimize the AI assistant effectiveness
