UI widgets
==========

To help you build a dynamic and interactive UI for your AI assistant, Alan AI provides UI widgets.

UI widgets are user interface elements designed to enhance the functionality of voice- and text-based AI assistants. They serve as interactive components within the AI assistant interface, allowing users to receive information and perform actions in an intuitive and engaging way.

In Alan AI, you can use the following UI widgets for your AI assistants:
